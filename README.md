## 2. Prompt đã sử dụng

### Prompt 1:
Hãy chỉnh sửa mã HTML để tạo giao diện giống trang Apple:
- Có thanh menu trên cùng
- Có tiêu đề lớn ở giữa
- Có mô tả ngắn
- Có 2 nút (Learn more, Buy)
- Có hình ảnh sản phẩm

### Prompt 2:
Hãy viết CSS để tạo giao diện giống mẫu Apple:
- Căn giữa nội dung
- Font chữ đơn giản, hiện đại
- Nút bo tròn
- Màu sắc giống Apple (xanh, trắng, xám)
- Giao diện tối giản
